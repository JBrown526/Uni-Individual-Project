Instructions to run:

1. Build project using Gradle

2. Using nextstepsdb schema.ddl create and host a MySQL server, an easy-to-use GUI is available in the form of MySQL Workbench

3. Set the database username and password in the application.properties file under resources

4. Run project

5. Access localhost:8080 to see available pages: index and login

6. Using a tool such as POSTMAN, HTTP requests can be sent to and from the address domain
